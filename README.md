# 长安志 - 数字文化展馆

> 以数字之笔，书长安千年

一个展示西安历史文化的数字展馆网站，包含文化解码、探访实录、风物档案等板块。

## 项目结构

```
.
├── index.html          # 卷首页面
├── culture.html        # 文化解码首页
├── culture-detail.html # 文化解码详情页
├── visit.html          # 探访实录首页
├── visit-detail.html   # 探访实录详情页
├── archives.html       # 风物档案首页
├── archives-detail.html # 风物档案详情页
├── map.html            # 长安舆图页面
├── css/                # 样式文件
├── js/                 # JavaScript文件
├── images/             # 图片资源
└── data/               # 数据文件
```

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)

## 运行方式

直接打开 `index.html` 即可浏览网站，无需服务器环境。

## 功能模块

1. **卷首** - 网站入口，展示长安主题视觉
2. **文化解码** - 解读长安历史文化
3. **探访实录** - 沉浸式探访体验
4. **风物档案** - 记录长安美食与非遗
5. **长安舆图** - 古地图交互展示

## 素材来源

图片素材来源：小红书博主@长安摄影师、@古都漫步、@丝路影像、@城墙记忆

文字内容参考：《长安十二时辰》、《西安通史》、陕西历史博物馆官网、西安市文旅局
